Sistema de Autenticación y Estadísticas

Sistema de Login

Autenticación de 3 factores implementada:

  • Usuario y Contraseña: Credenciales por defecto
  • Código de Acceso: Verificación adicional (WEB2024)
  • Captcha Matemático: Operación aleatoria simple
  • Sesión Persistente: Usa localStorage del navegador

Contador de Visitas

Sistema que cuenta cada acceso único a la aplicación:

  • Almacenamiento Local: Usa localStorage para persistencia
  • Contador Global: Incrementa con cada carga de página
  • Visualización en Tiempo Real: Mostrado durante login y sesión
  • Persistencia entre Sesiones: Los datos no se pierden al cerrar

Almacenamiento de Datos

Tecnologías utilizadas para guardar información:

  • localStorage: Para sesión y contador de visitas
  • Cookies: Alternativa para sesiones prolongadas
  • Session Storage: Datos temporales de la sesión
  • Sin Backend: Todo funciona en el cliente
